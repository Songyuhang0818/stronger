## 1.prototype

每一个函数都有一个 prototype 属性，这个属性指向函数的原型对象

```js
function Person() {}
Person.prototype.name = "songyuhang";

const person1 = new Person();
const person2 = new Person();
console.log(person1.name); // songyuhang
console.log(person2.name); // songyuhang
```
